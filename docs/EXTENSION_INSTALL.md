# Chrome插件版 - 安装教程

## 什么是Chrome插件?

Chrome插件(Chrome Extension)是浏览器的扩展程序,可以增强浏览器功能。本工具提供Chrome插件版本,无需安装Tampermonkey,更加轻量便捷。

## 兼容性

本插件支持以下浏览器:
- ✅ Chrome (推荐)
- ✅ Microsoft Edge
- ✅ Opera
- ✅ Brave
- ✅ 其他基于Chromium的浏览器

## 安装方式

### 方式一:开发者模式安装(推荐)

#### 第一步:下载插件

1. 下载项目源码
   ```bash
   git clone https://github.com/weiruankeji/weiruan-Netdisk.git
   ```
   或直接下载ZIP压缩包并解压

2. 找到插件目录: `src/extension/`

#### 第二步:加载插件

**Chrome浏览器:**
1. 打开Chrome浏览器
2. 地址栏输入: `chrome://extensions/`
3. 右上角开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `src/extension/` 文件夹
6. 点击"选择文件夹"

**Edge浏览器:**
1. 打开Edge浏览器
2. 地址栏输入: `edge://extensions/`
3. 左下角开启"开发人员模式"
4. 点击"加载解压缩的扩展"
5. 选择 `src/extension/` 文件夹
6. 点击"选择文件夹"

#### 第三步:固定插件图标(可选)

1. 点击浏览器右上角的拼图图标(扩展程序)
2. 找到"网盘直链下载增强工具"
3. 点击图钉图标,将其固定到工具栏

### 方式二:打包安装

#### 第一步:打包插件

1. 在Chrome中打开 `chrome://extensions/`
2. 开启"开发者模式"
3. 点击"打包扩展程序"
4. 扩展程序根目录选择: `src/extension/`
5. 点击"打包扩展程序"
6. 会生成 `.crx` 文件

#### 第二步:安装.crx文件

1. 将生成的 `.crx` 文件拖拽到扩展程序页面
2. 点击"添加扩展程序"
3. 等待安装完成

**注意:** 某些浏览器可能禁止直接安装.crx文件,建议使用方式一。

## 验证安装

### 检查插件是否加载

1. 打开 `chrome://extensions/`
2. 查看是否有"网盘直链下载增强工具"
3. 确认状态为"已启用"

### 测试功能

1. 打开百度网盘或其他支持的网盘
2. 查看页面上是否出现"直链下载"按钮
3. 点击插件图标,查看弹出窗口

## 使用方法

### 基本使用

1. **访问支持的网盘网站**
   - 百度网盘、天翼云盘、蓝奏云等

2. **选择要下载的文件**
   - 某些平台需要勾选文件

3. **点击"直链下载"按钮**
   - 自动获取直链并下载

4. **查看下载进度**
   - 在浏览器下载栏查看

### 插件弹窗功能

点击插件图标,可以看到:

#### 支持平台
显示所有支持的网盘平台

#### 统计信息
- 总下载数
- 今日下载数

#### 设置选项
- **显示通知**: 下载时显示浏览器通知
- **保存下载历史**: 记录下载历史

#### 清除历史
点击"清除下载历史"按钮,清空统计数据

## 高级功能

### 批量下载

1. 选择多个文件
2. 点击"直链下载"
3. 在弹窗中选择要下载的文件
4. 点击"批量下载全部"或单独下载

- Chrome下载管理器

- Free Download Manager
- 其他支持Chrome下载API的下载器

## 常见问题

### 1. 插件无法加载?

**解决方法:**
- 确认已开启"开发者模式"
- 检查文件夹路径是否正确
- 查看控制台错误信息
- 重新加载插件

### 2. 插件已加载但不工作?

**解决方法:**
- 刷新网盘页面
- 检查插件是否已启用
- 查看浏览器控制台错误(F12)
- 检查网盘账号是否已登录

### 3. 下载失败?

**解决方法:**
- 检查文件是否可以正常下载
- 确认网盘账号已登录
- 某些大文件可能需要会员
- 检查网络连接

### 4. 图标不显示?

**解决方法:**
- 图标文件可能缺失
- 暂时不影响使用
- 可以通过扩展程序页面访问

### 5. 获取直链后下载慢?

**原因:**
- 网盘服务器限速
- 网络环境问题

**解决方法:**
- 使用下载管理器(如IDM)
- 开通网盘会员加速
- 更换网络环境

## 更新插件

### 自动更新
如果插件发布到Chrome应用商店,会自动更新。

### 手动更新
1. 下载最新版本源码
2. 在 `chrome://extensions/` 页面
3. 点击插件的"重新加载"按钮
4. 或删除旧版本,重新加载新版本

## 卸载插件

1. 打开 `chrome://extensions/`
2. 找到"网盘直链下载增强工具"
3. 点击"移除"按钮
4. 确认删除

## 权限说明

插件需要以下权限:

- **storage**: 保存设置和统计数据
- **downloads**: 触发浏览器下载
- **webRequest**: 拦截和修改网络请求(获取下载链接)
- **host_permissions**: 访问支持的网盘网站

**隐私保护:**
- 所有数据仅保存在本地
- 不上传任何信息到服务器
- 不收集用户隐私数据

## 开发调试

1. 打开网盘页面
2. 按F12打开开发者工具
3. 切换到"Console"标签
4. 查看脚本日志输出

### 调试后台脚本

1. 打开 `chrome://extensions/`
2. 找到插件,点击"背景页"或"service worker"
3. 在打开的窗口中查看后台日志
